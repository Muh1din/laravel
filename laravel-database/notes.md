# DB Facade
[x] DB Facade diguanakan untuk berkomunikasi dengan database.


# database comment
[x] php artisan db = untuk masuk ke database configurasi laravel
[x] php artisan db:table = untuk melihat table yang ada di database saat ini
[x] php artisan db:show = untuk melihat informasi database
[x] php artisan db:monitor = untuk monitoring jumlah koneksi yang ada di database
[x] php artisan db:seed = untuk menambah data yang ada di database, [untuk seeder]
[x] php artisan db:wipe = untuk menghapus selutuh table yang ada di database
